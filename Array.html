Array

Array is a Data Structure that can store multiple value at a time.

Array Declaration :-

Syntax :-

dataType   arrayName[SIZE]

dataType->any valid C data type

arrayName->any validCidentifier

SIZE->Integer Literal/Constant,it can't be variable

Example :-

  1. int data[10];

    2. here; an array named 'data' is created of capacity 10 of integer type. i.e data variable can store 10 value of int type at a time.

  2. float m[20];

    3. here; m is an array,that can Store 20 float value at a time.

  3. x[100];

    4. here; x is a an array,that can store 100 int value at a time.
